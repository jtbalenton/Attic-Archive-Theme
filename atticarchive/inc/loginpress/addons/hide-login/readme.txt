== Changelog ==

= 1.2.3 - 2020-11-09 =
* Compatibility: LoginPress - Hide Login is compatible with <a href="https://wordpress.org/plugins/translatepress-multilingual/">TranslatePress</a> plugin now.
* Compatibility: LoginPress - Hide Login is compatible with LoginPress - Limit Login Attempts Add-On now.

= 1.2.2 - 2020-06-24 =
* BugFix: PHP 7.1 Fatal error fix.

= 1.2.1 - 2020-01-16 =
* Compatibility: LoginPress - Hide Login is compatible with French language now.

= 1.2.0 - 2019-11-15 =
* Enhancement: Added a filter `loginpress_hide_login_email_subject` to customize the email subject that send on set/reset the login slug.
* Enhancement: Added a filter `loginpress_hide_login_email_notification` to customize the email content that send on set/reset the login slug.

= 1.1.4 - 2019-10-23 =
* BugFix: Sanitize email addresses input.
* Enhancement: Added a filter `loginpress_hidelogin_wp_admin_redirect` to apply redirect on /wp-admin (By default - Redirects on 404).
* Enhancement: Code refactoring.

= 1.1.3 - 2019-08-10 =
* BugFix: Sanitize Email Address field.
* Compatibility: LoginPress - Hide Login is compatible with Arabic language now.

= 1.1.2 - 2019-08-02 =
* BugFix: Redirect to 404 on wp-admin.

= 1.1.1 - 2019-07-16 =
